# Changelog

All notable changes to the "artifact-hub" extension will be documented in this file.

## [0.1.0] - 2025-01-15

### Added
- Initial release
- Search and discover artifacts across multiple repositories
- Support for GitHub, GitLab, and generic Git platforms
- One-click installation of chat modes, instructions, prompts, and tasks
- Conflict detection and resolution
- Dependency resolution
- Update notifications
- Private repository support with authentication
- Status bar integration
- Auto-refresh catalogs

### Features
- Full-text search with intelligent ranking
- Filter by type, language, framework, category, and difficulty
- Preview artifacts before installing
- Manage installed artifacts
- Profile support for bulk installations

### Supported Artifact Types
- Chat Modes (`.chatmode.md`)
- Instructions (`.md`)
- Prompts (`.md`)
- Tasks (`.md`)
- Profiles (`.json`)
